# Landing Page

A minimal, performant landing page for `tarun.vrma.net`.

## Structure

- `index.html`: Main entry point with semantic HTML structure.
- `style.css`: Minimal CSS with dark mode support.

## Local Development

To run the site locally:

```bash
python3 -m http.server
```

Then visit `http://localhost:8000` (or 8080 depending on the command).

## Deployment

The site is ready for deployment on Cloudflare Pages.
